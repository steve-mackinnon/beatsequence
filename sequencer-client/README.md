# audio-sequencer

## Dependencies

- [Yarn](https://yarnpkg.com/): `brew install yarn`
- [Parcel](https://parceljs.org/getting-started/webapp/): `yarn add --dev parcel`

## Running local server

1. Run `yarn install`
2. Run `yarn start`
3. Navigate to [http://localhost:1234](http://localhost:1234)
